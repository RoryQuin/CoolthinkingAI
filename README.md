# 项目简介
一个面向思想政治教育师范生的马克思主义基础概念条目库。<br>

这是一个结构化的条目库。它收录了258个马克思主义基础概念条目，并为每个条目设置了元数据。它用于为思想政治教育师范生训练课程代理解决方案提供预训练知识库，也可以下载到个人阅读器中浏览。
